# LineBotTest
Use Docker compose and ubuntu to run it


# Installation
Ubuntu
```bash
sudo apt install docker-desktop
git clone https://github.com/flier268/LineBotTest.git
cd LineBotTest
make
```
**You need to edit config.yml, and fill line channel secret and token!**
